Textbox 文本输入框组件 Demo

PhoneNumberTextBox 电话号码输入框

  • 支持手机号、固定电话号码，通过accept属性选择支持类型
  • 手机号码是1开头的11位数字
  • 固定电话号码是0开头的10-12位数字（3-4位区号+7-8位电话号码）

默认支持手机号和固定电话号码

code
                

                    
                

            
                    KISSY.use('gallery/form/1.3/textbox/index', function (S, TextBox) {
                        var textbox = new TextBox.PhoneNumberTextBox('#textbox');
                        textbox.render();
                        textbox.on('blur', function () {
                            this.checkValidity();
                        });
                        textbox.on('invalid', function () {
                            alert(this.get('validationMessage'));
                        });
                    });
