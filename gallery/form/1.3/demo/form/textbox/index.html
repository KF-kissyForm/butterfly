Textbox 文本输入框组件

  • 使用
  • 内置功能
  • 类型
  • 样式

作者：牧云

Textbox是一个基于kissy1.2的文本输入框封装组件。

使用

HTML

JavaScript

配置包
                KISSY.config({
                    packages: [{
                        name:"gallery",
                        path:"../../../../../../",
                        charset:"utf-8"
                    }]
                });
实例化
                KISSY.use('gallery/form/1.3/textbox/index', function (S, TextBox) {
                var textbox = new TextBox('#textbox');
                textbox.render();
             })

内置功能：

  • 聚焦时自动全选文本，默认关闭；
  • 禁用粘贴，默认关闭；
  • 内容改变时（例如键入、粘贴、动态赋值等操作），自动去除左右空格（trim），默认开启；
  • 动态改变maxLength时，截断多余内容；
  • 兼容Placeholder行为。原生的Placeholder功能会在输入框键入内容的时候才会隐藏Placeholder，模拟的目前只会在focus的时候隐藏Placeholder。可以选择统一使用模拟的实现方式；
  • 封装了The Constraint Validation API，兼容required和pattern属性；
  • 输入内容的时候显示一个弹层，根据输入内容动态显示信息，默认关闭。

扩展类型：

  • 号码输入框，只允许输入数字；
  • QQ号输入框，有效的QQ号是4-10位的数字；
  • 电话号码输入框，可以选择支持的号码类型，手机号码，固定电话号码，还是两者皆可。有效的手机号码是11位的数字。有效的固定电话号码必须包含区号（3-4位），加上7-8位的号码，总长度在10-12位之间。
